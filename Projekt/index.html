Coronavirus

Ab China hat sich das neue Coronavirus SARS-CoV-2 weltweit verbreitet.Die Stadt Wuhan in der Provinz Hubei war das Zentrum des Ausbruchs.

Die verwandte Krankheit heißt Coronavirus Disease 2019 (COVID-19).Innerhalb kurzer Zeit, nachdem das SARS-CoV-2-Virus im Dezember 2019 erstmals entdeckt wurde, breitet es sich sehr effektiv von Person zu Person aus, insbesondere durch Tröpfcheninfektion und Aerosolinfektion.

Sie können die COVID-19-Krankheit durch das Corona-Virus bekommen.

Jede Krankheit kann anders sein.

So erkennt man selbst die Krankheit COVID-19:
  • Jemand wird sich erkälten.
  • Sie können husten.
  • Sie können Fieber haben.
  • Möglicherweise können Sie für eine Weile keinen Geruch riechen.
  • Möglicherweise haben Sie Schwierigkeiten beim Atmen.
  • Möglicherweise können Sie vorerst nichts probieren.

    • Wer wird wann gegen COVID-19 geimpft ?

    Zu Beginn stehen nur begrenzte Impfstoffmengen zur Verfügung. Daher können nicht alle Menschen direkt geimpft werden.

    In der Impfverordnung hat das Bundesgesundheitsministerium festgelegt, wer zuerst geimpft wird. Es gibt drei Stufen der Priorisierung.

    In den Bundesländern kann es abweichende Regelungen geben. Zudem können die Bundesländer innerhalb der drei Impfgruppen eine weitere Reihenfolge festlegen.

    Das Bundesgesundheitsministerium rechnet damit, dass allen Menschen bis Ende Sommer 2021 ein Impfangebot gemacht werden kann.
